# 🎯 One-Line Commands to Run the Project

## Fastest Way to Start Everything

### **Windows PowerShell / CMD / Terminal:**
```bash
python start_all.py
```

### **If Python script fails, try Windows Batch:**
```bash
start_all.bat
```

### **Or use PowerShell directly:**
```powershell
.\start_all.ps1
```

---

## What You'll See

1. A new window opens: **FastAPI Backend** (Port 8000)
2. Another window opens: **Vite Frontend** (Port 5173)
3. Your browser automatically opens to `http://localhost:5173`

---

## Stopping Everything

- **Press `Ctrl+C`** in the main terminal window, OR
- **Close the backend and frontend windows** that opened

---

## URLs After Starting

| What | URL |
|------|-----|
| 🎨 Frontend App | `http://localhost:5173` |
| 🔌 Backend API | `http://localhost:8000` |
| 📖 API Docs | `http://localhost:8000/docs` |

---

**That's it! Everything runs perfectly with just one command.** ✨
